## Project Overview

This assessment focuses on cleaning, transforming, and analyzing retail sales data using Python and Power BI.

The dataset contained several data quality issues including missing values, inconsistent categories, mixed date formats, invalid quantity values, and exposed customer information. These issues were identified and resolved before generating business insights and dashboards.

## Data Quality Issues Addressed

* Missing prices corrected using product master data
* Category inconsistencies standardized
* Product naming standardized
* Mixed date formats converted to datetime format
* Invalid quantity records removed
* Customer information masked for privacy protection
